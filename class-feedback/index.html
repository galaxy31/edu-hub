버튼을 눌렀을 때 권한 요청이나 로그인 화면이 뜨면, 해당 도메인 계정으로 로그인되어 있는지 확인하세요. 다른 도메인 계정으로는 열리지 않습니다. 처음 한 번은 권한 허용이 필요합니다.
“현재 파일을 열 수 없습니다 / 주소를 확인한 다음 다시 시도해 주세요” 같은 Google Drive 화면이 뜨나요? 이는 페이지 오류가 아니라, 버튼과 다른 도메인 계정으로 로그인된 상태라는 뜻입니다. 누른 버튼에 맞는 계정(@sc.gyo6.net 또는 @aiep.gbe.kr)으로 다시 로그인한 뒤, 이 페이지에서 버튼을 다시 눌러주세요. 여러 구글 계정을 함께 쓰는 경우, 브라우저 오른쪽 위 프로필에서 계정을 전환하거나 시크릿 창에서 해당 계정으로만 로그인하면 가장 확실합니다.

처음 사용하는 선생님께

  1. 위에서 본인 계정에 맞는 버튼을 누릅니다.
  2. 처음엔 권한 허용 화면이 한 번 나옵니다 → 허용을 누릅니다.
  3. 화면 맨 위에서 Gemini API 키를 한 번 입력·저장합니다. 이후에는 다시 넣지 않아도 됩니다(본인 계정에만 저장).
  4. 수업 → 과제 → 채점기준표를 고르고 채점 시작을 누르면 됩니다.
Gemini API 키는 Google AI Studio에서 무료로 발급해 복사·붙여넣기 하면 됩니다. 키 확인·삭제 버튼은 앱 화면 상단에 있습니다.
