“현재 파일을 열 수 없습니다 / 주소를 확인한 다음 다시 시도해 주세요” 같은 Google Drive 화면이 뜨나요? 이는 페이지 오류가 아니라, 버튼과 다른 도메인 계정으로 로그인된 상태라는 뜻입니다. 누른 버튼에 맞는 계정(@sc.gyo6.net 또는 @aiep.gbe.kr)으로 다시 로그인한 뒤, 이 페이지에서 버튼을 다시 눌러주세요. 여러 구글 계정을 함께 쓰는 경우, 브라우저 오른쪽 위 프로필에서 계정을 전환하거나 시크릿 창에서 해당 계정으로만 로그인하면 가장 확실합니다.

처음 사용하는 선생님께

  1. 위에서 본인 계정에 맞는 버튼을 누릅니다.
  2. 처음엔 권한 허용 화면이 한 번 나옵니다 → 허용을 누릅니다. (“확인되지 않은 앱” 경고가 뜨면 고급 → 이동을 눌러 진행)
  3. 왼쪽 설정에서 AI 제공사를 고르고 API 키를 한 번 입력·저장합니다. 이후에는 다시 넣지 않아도 됩니다(본인 계정에만 저장).
  4. 수업 목록 불러오기 → 수업·과제 선택 → ① 제출물 스캔 → ② 전체 세특 생성 순서로 진행합니다.
  5. 생성된 내용을 검토·수정한 뒤 ③ 시트로 저장을 누르면, 선택한 수업 이름으로 시트가 만들어져 저장됩니다.
Google Gemini는 Google AI Studio에서 무료 등급으로 API 키를 발급받아 시작할 수 있습니다. Claude·OpenAI도 선택할 수 있으며, 각 제공사의 키 발급 방법은 앱 화면의 안내를 참고하세요. 키 보기·저장 버튼은 앱 화면 왼쪽에 있습니다.

세특 작성 규칙 안내

기본 작성 지침에는 다음 규칙이 이미 반영되어 있습니다.

  1. 공백 포함 500자를 넘지 않도록 자동으로 관리됩니다.
  2. 학생 이름·반·번호 등 인적사항은 본문에 들어가지 않습니다.
  3. “학생은~” 같은 주어 지칭 없이 행위·역량을 바로 서술합니다.
  4. 강점과 성장 등 긍정적 내용만 담고, 부정적·개선요구 표현은 넣지 않습니다.
